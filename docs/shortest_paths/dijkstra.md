# Dijkstra

Computes shortest paths from a source vertex in graphs with non-negative edge weights.

---

## Complexity

```
O(E log V)
```

---

## C++

```cpp
auto result = graph.dijkstra(0);
```

---

## Python

```python
graph.dijkstra(0)
```

---

## Applications

- GPS navigation
- Network routing
- Path planning
