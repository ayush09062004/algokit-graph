# AlgoKit Graph

A modern C++17 graph algorithms library with Python bindings.

## Features

- BFS
- DFS
- Connected Components
- Strongly Connected Components
- Cycle Detection
- Topological Sort
- Dijkstra
- Bellman-Ford
- Floyd-Warshall
- Kruskal
- Prim
- Disjoint Set Union

See the navigation on the left to explore the documentation.

See [`BUG_REPORT.md`](https://github.com/ayush09062004/algokit-graph/blob/main/BUG_REPORT.md)
