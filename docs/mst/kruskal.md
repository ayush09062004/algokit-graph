# Kruskal's Algorithm

Computes a Minimum Spanning Tree using edge sorting and Disjoint Set Union.

---

## Complexity

```
O(E log E)
```

---

## C++

```cpp
auto mst = graph.kruskal();

mst.total_weight();
```

---

## Python

```python
mst = graph.kruskal()
```

---

## Applications

- Network design
- Road construction
- Clustering
